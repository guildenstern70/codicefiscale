# Codicefiscale

This program computes the Italian fiscal code.

## Setup

To run this program you to need setup an Erlang+Elixir environment, as it is described here:
https://elixir-lang.org/install.html

This program uses the exqlite library to interact with a SQLite database.

### Download dependencies

    mix deps.get

### Build & Run

    mix run

## Run tests

    mix test
